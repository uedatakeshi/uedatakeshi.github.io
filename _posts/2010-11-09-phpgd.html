もう毎回の事なんだからさあ～って気もしないでもないが、やり方を教わったのでメモ。

サーバーに入って、rootになったら、
# yum search php
と打つ。
するとだーっとリストが出る。
その中に
php-gd.i386 : gd グラフィックライブラリを使用するための PHP アプリケーション用モジュール
こんなのがある。
それをこうする。
# yum install php-gd.i386

途中でyesかnoか聞かれるので、yesすると
Installed:
  php-gd.i386 0:5.1.6-27.el5

Complete!
というように入る。
あとは、
# /etc/init.d/httpd restart
再起動する。
